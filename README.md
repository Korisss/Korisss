# Hi, I'm Pavel

Cybersecurity Architect and Security Engineer working at the intersection of security architecture, DevSecOps, cloud-native infrastructure and software engineering.

My work combines architecture with hands-on engineering. I design security solutions, build automation, develop tools in Go, and work with modern cloud-native platforms to make security practical, scalable and integrated into engineering workflows.

I'm particularly interested in infrastructure security, Kubernetes, platform engineering and the growing role of AI in security engineering.

## Engineering Principles

I believe security should be:

* automated whenever possible;
* simple enough to operate effectively;
* built into systems instead of added afterwards;
* closely integrated with engineering processes;
* driven by practical risk reduction rather than compliance alone.
